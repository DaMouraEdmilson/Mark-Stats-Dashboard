# Market Stats Dashboard + Databricks Genie Spaces

**End-to-end Databricks analytics pipeline** I built as Analytics Engineer at Newmark. 

This became a **flagship product** for the National Research team.

### 🎯 Business Impact
- Reduced analyst manual effort by **up to 80%**
- Powers national Market Stats reporting across Office and Industrial sectors
- Feeds interactive Sigma dashboards and **Databricks Genie Spaces** (AI conversational analytics)

### What The Pipeline Does
- Automated ingestion of CoStar data from SharePoint (Industrial + Office + Leasing Activity)
- Complex market rollups, weighted rents, absorption, deliveries, and property size bucketing
- Clean data model optimized for Databricks Genie and Excel/ODBC connections

### Key Technical Features
- Configurable market normalization (single source of truth)
- Robust SharePoint automation + validation
- Leasing Activity integration with mismatch warnings
- Production-grade Delta Lake pipeline with Window functions

### Tech Stack
- **Databricks** (PySpark, Delta Lake, SQL, Genie Spaces)
- Python + Spark ETL
- Sigma dashboards
- Advanced SQL & Data Modeling
